ABOUT ME

Hello! I'm Max, a game developer and programmer from England. I've been developing games since I was 15, and designing games for even longer. I'm experienced in a range of languages and programs (as seen below), but I'm always looking for areas to improve and things to learn.

I got a Distinction* studying game design and programming in college where I developed various games in Unity using C#, before pivoting over to Godot to make use of its open-source framework. I've dabbled in creating my own game engine in C++ although there should be more to say about that soon...

RECENT GAME

Midnight transmission is a 2D top-down exploration game about a surreal interconnected world; explore abstract and absurd areas collecting tools that can change the environment or improve exploration.

The game was developed in Godot with GDScript by a team of three core developers and a small team of collaborative developers. The unique visual style comes from how much we've capitalised on the game as a collaborative effort, using composers, artists, and developers with a wide range of aesthetics.

For more information and a link to the game go to the portfolio page.
